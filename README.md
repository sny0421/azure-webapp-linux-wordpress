# azure-webapp-linux-wordpress

## 概要
Azure App Service で WordPress を実行するための環境を構築するテンプレートです。

今後 Windows で PHP 8 がサポートされなくなるため、Linux で WordPress を実行した上で必要なデータベースなども準備しています。
